# mentoring-project
This is educational project.
The goal is to create a simple Java web application and then build it using [Gradle](https://gradle.org/).
Provided deployment capabilities of web applications with an embedded Tomcat web container.
## Build
* Mac OS ```./gradlew build```
* Windows ```gradlew build ```
## Deploy
* Mac OS ```./gradlew tomcatRunWar```
* Windows ```gradlew tomcatRunWar ```
